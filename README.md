🧮 Calculator Web Application
Level 2 – Task 1
📌 About the Project
This project is a simple and interactive Calculator Web Application built using HTML, CSS, and JavaScript.
The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It features a clean layout, responsive design, and dynamic functionality using JavaScript.
🎯 Task Objective
The objective of this task is to:
1)Create a functional calculator interface
2)Implement arithmetic operations
3)Handle user input dynamically
4)Display results instantly
Use JavaScript event listeners and conditional logic
🚀 Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
🧹 Clear (C) button
🖥️ Interactive display screen
📱 Responsive and user-friendly design

🛠️ Technologies Used
HTML5 – Structure of the calculator
CSS3 – Styling and layout alignment
JavaScript – Logic and functionality

⚙️ How It Works
1)Each button is connected using JavaScript event listeners.
2)User inputs are displayed on the screen.
3)Arithmetic operations are processed using JavaScript operators.
4)The result is calculated and displayed when "=" is clicked.

▶️ How to Run the Project
1)Download or clone this repository.
2)Open the project folder.
3)Open index.html in any web browser.
4)Start using the calculator.
